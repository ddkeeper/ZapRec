# 2026/3/18
## round 1
1. 鼠标悬停在小窗内部时仍旧无法拖拽移动小窗位置
2. 画中画圆形时拉四个点，有的时候是等比例拉升，有的时候又会拉成椭圆，需要固定为等比例伸缩
3. 录制过程中应该支持高亮显示信息源录制状态，打开即高亮，关闭即灰色，只是无法修改标记为不可用；
4. 目前把鼠标移入画中画区域，形状按钮就会显示，这个时候如果同时把鼠标移动到伸缩点位执行伸缩就会无响应，即形状按钮出现时，伸缩操作是无效的，即使鼠标显示为了双箭头
5. 要不要把圆形时的四个点位设置在左上，左下，右上和右下这四个位置，方便用户更好地进行等比例缩放
6. 圆形状态下仍旧能看到周边淡色的四个矩形角，且用户把鼠标放在上面还可以进行缩放，严重影响体验
7. 处于区域录制过程中，鼠标在录制源开关区移动时形状会频繁刷新变化，在禁止图标和正常光标之间快速切换，出现那种非常不自然的闪烁，鼠标在画中画边缘区域移动时也会出现类似的闪烁，鼠标频繁在缩放图标与光标间切换；而在全屏录制模式下录制却不会出现类似的问题
- [x] 打开画中画以后，关闭主窗口，即主工具条栏后画中画窗口不会一并关闭，影响用户体验
